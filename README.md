# Claude Chat Logs

Этот репозиторий автоматически сохраняет историю чатов с Claude каждые 5 минут.

## Структура

- `chat-logs/` — папка с сохранёнными сессиями чата
  - Файлы формата: `chat_YYYY-MM-DD_HH-MM.md`

## Настройка

Автосохранение настроено через планировщик задач Windows и Claude Code hook.
